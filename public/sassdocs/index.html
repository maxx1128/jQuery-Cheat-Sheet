    keyframes

    @mixin keyframes($animationName) { ... }

    Description

    This is the main mixin for creating keyframe animations with cross-browser compatibility automatically set up. It needs the name and also the body with the keyframe locations and properties.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $animationName

    The name of the animation to be referenced

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    @include keyframes(name-here) {
   0%   { color: black; }
   50%   { color: white; }
   100% { color: black; }
}

@include animation('name-here 1s linear 0s infinite');

    variables

    breakpoints

    $breakpoints: (
  mobile: new-breakpoint(min-width 320px),
  mobile-h: new-breakpoint(min-width 480px),
  tablet: new-breakpoint(min-width 787px),
  desktop: new-breakpoint(min-width 900px),
  huge: new-breakpoint(min-width 1200px),
);

    Description

    These are the core breakpoints for the grid. They use the values in the breakpoint lengths map to determine the breakpoints. This, along with other grid functionality, uses the Neat dependency.

    Type

    Map

    Example

    body {
  @include media(breakpoint(tablet)) {
    @include span-columns(10);
    @include shift(1);
  }
}

    Used by

    • [function] breakpoint

    Links

    • http://neat.bourbon.io/

    fix-flex

    @mixin fix-flex($fix-direction, $fix-width, $fix-margin) { ... }

    Description

    The fix-cell and flex-cell mixins included components of the fix-flex mixin, which is used for items with a fixed width on one side and flexible on the other.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $fix-direction

    If the fixed element is on the left or right side

    		String—none
    $fix-width

    The width of the fixed element

    		Length—none
    $fix-margin

    The margin length between the two items

    		Length—none

    Example

    @include fix-flex('left', 2em, 10px);

    Requires

    • [mixin] fix-cell
    • [mixin] flex-cell

    Author

    • Krispin Dolbear, EMA

    The font size that all other element font sizes are scaled off of. This is often the size of basic paragraph text, and is scaled up for headers.

    $base-font-scale: 0;

    modular-size-map

    $modular-size-map: (
   h1: $base-font-scale + 6,
   h2: $base-font-scale + 5,
   h3: $base-font-scale + 4,
   h4: $base-font-scale + 3,
   h5: $base-font-scale + 2,
   h6: $base-font-scale + 1,
   p: $base-font-scale,
);

    Description

    This uses the Bourbon modular-scale function to create the structure of how fonts will be sized in the modular scale. An each loop is used to generate the scaled font sizes.
